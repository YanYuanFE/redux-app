# redux-app
a simple redux && react-redux

> 动手实现一个简单的redux、react-redux

### 使用Redux实现简单计数器


``` bash
// clone repo
git clone https://github.com/YanYuanFE/redux-app.git


cd redux-app

// checkout branch
git checkout part-1

// install
npm install

// start
npm start

```
